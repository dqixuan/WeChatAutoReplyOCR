# WeChatAutoReplyOCR
WeChatAutoReplyOCR is a Python-based automation tool for WeChat that can automatically detect unread messages which runs on linux(e.g ubuntu) and reply to them. It leverages OCR (EasyOCR) to read message content and uses PyAutoGUI for safe mouse interactions, even on locked screens or after screen state changes.
